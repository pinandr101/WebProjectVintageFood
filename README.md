### Vintage Food

Vintage Food - уникальная сеть ресторанов с незабываемыми блюдами. Хотите посмотреть меню, найти контакты компании или же заказть доставку на дом? Теперь всё это возможно сделать с помощью данного сайта!

!Проект выполнен в образовательных целях на основе макета, расположенного в открытом доступе на сайте Figma. Все названия и личные данные, представленные на сайте, выдуманы, а совпадения случайны.

## Описание

Один из первых мини-проектов в сфере веб-разработки. Целью работы являлось освоение базовых навыков пользования языком разметки html и основами css. По этой причине на сайте представлен практически нулевой функционал.

## Установка
Для запуска проекта достаточно выполнить одно из двух:
- Опробовать интернет-ресурс через ссылку на git-deploy
- Клонировать репозиторий и открыть файл "index.html"
